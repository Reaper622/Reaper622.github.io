CSS float

float 设计初衷

用来实现文字环绕效果

包裹与破坏

包裹:

  1. 收缩
  2. 坚挺
  3. 隔绝(对外无影响)

破坏:

父级元素高度塌陷

如果清除浮动(带来的影响)
  1. 脚底插入clear:both;属性的盒子 //但外部仍有影响
  2. 父元素BFC(IE8+)或者haslayout(IE6/IE7) //外部不会对内部有影响
float滥用
  1. 元素block块状化
  2. 破坏性造成的紧密排列特性(去空格化)
块妆化的危害:
  • 容错性差，易出问题。
  • 需要元素固定尺寸，很难重复使用。
  • 低版本IE兼容性低。
float与流体布局
实现文字单侧固定
智能自适应布局
float
display:table-cell; IE8+
display:inline-block; IE7

float兼容性

让IE7用户落泪的浮动问题
  • 含clear的浮动元素包裹不正确的问题
  • 浮动元素倒数2个莫名垂直间距问题
  • 浮动元素最后一个字符重复问题
  • 浮动元素楼梯排列问题
  • 浮动元素和文本不在同一列的问题

使用Float需谨慎！
